# Authentication API

Backend authentication APIs built using Node.js, Express and PostgreSQL.

## Features
- User Registration API
- User Login API
- Password hashing using bcrypt
- PostgreSQL database integration
- Clean MVC folder structure

## Tech Stack
- Node.js
- Express.js
- PostgreSQL
- bcrypt
- dotenv

## API Endpoints

POST /api/auth/register  
POST /api/auth/login
